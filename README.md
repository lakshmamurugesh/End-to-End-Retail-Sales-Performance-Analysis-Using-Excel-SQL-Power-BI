# End-to-End Retail Sales Performance Analysis  
**Using Excel, SQL & Power BI**

---

## 1. Introduction
This project presents an end-to-end analysis of retail sales data to evaluate business performance and uncover actionable insights. It integrates data cleaning, SQL-based analysis, KPI tracking, and interactive visualization using Power BI.

---

## 2. Business Problem Statement
Retail businesses often struggle to track sales trends, profitability, and customer behavior across regions and product categories. This project aims to address these challenges by providing data-driven insights to support strategic decision-making.

---

## 3. Project Objectives
- Analyze overall sales and profit performance  
- Identify high- and low-performing products and regions  
- Track key performance indicators (KPIs)  
- Understand customer purchasing behavior  
- Build an interactive Power BI dashboard for stakeholders  

---

## 4. Dataset Description
The dataset contains retail transaction-level data, including:
- Order details (Order ID, Date)
- Product and category information
- Customer details
- Sales, quantity, discount, and profit
- Regional and segment data

## 8. SQL-Based Analysis
SQL queries were written to:
- Aggregate sales and profit metrics  
- Perform time-based trend analysis  
- Rank products, customers, and regions  
- Create views and tables for KPI reporting
